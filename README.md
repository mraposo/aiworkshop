# AI Workshop - OpenEdge ABL Project

This project demonstrates OpenEdge ABL (Advanced Business Language) development patterns and best practices for building business applications.

## Project Structure

- **src/business/** - Business entity classes and data access logic
- **src/** - UI window files and application entry points
- **.windsurf/** - Development workflow configurations
- **doc/** - Documentation and design patterns
- **dump/** - Database schema definitions

## Key Features

- Business Entity Pattern implementation
- Customer and Item management functionality
- Modern ABL class-based architecture
- Comprehensive documentation

## Getting Started

This project uses the OpenEdge development environment and follows the Business Entity Pattern for data access and business logic separation.
